## IAIPM Sponsorship App

The official GitHub repo for IAIPM Sponsorship App.

###### Install Composer dependences

```
composer install
```

###### Install Node dependences

```
yarn install
```

###### Run Backend

```
php artisan serve
```

###### Build assets and run Frontend

```
yarn dev
```
